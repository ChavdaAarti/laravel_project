# Laravel CRUD Project

## Description
This is a basic Laravel project where I implemented CRUD operations.

## Features
- Add new records
- View records
- Update records
- Delete records

## Technologies Used
- Laravel
- PHP
- MySQL
- HTML, CSS

## How to Run
1. Clone the repository
2. Run composer install
3. Setup .env file
4. Run php artisan migrate
5. Run php artisan serve
